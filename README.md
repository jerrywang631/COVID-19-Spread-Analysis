## 👥 Team & Acknowledgements

This project was a collaborative effort by Zhenming Liu, Ziheng Wang, Jialun Wei, and Ruocheng Wen for the CSC111 course project at UTSG. 

Together, our team successfully:
*   Designed and implemented a custom 6-level `Tree` data structure, parsing large-scale COVID-19 datasets from official Canadian and US sources to accurately sort and store the data.
*   Developed backend computation logic to calculate regional case densities and average growth rates, directly illustrating the extent of the pandemic across specific locations and timeframes.
*   Integrated the Plotly and Pandas libraries to build an interactive CLI tool and generate geographic heatmaps, better visualizing the pandemic's impact in Canada and the US.

Note: Because this project was originally submitted via MarkUs (U of T's internal grading portal), the original Git commit history and version control records are unavailable in this repository. However, the final codebase presented here represents the equal and joint effort of our entire team.
